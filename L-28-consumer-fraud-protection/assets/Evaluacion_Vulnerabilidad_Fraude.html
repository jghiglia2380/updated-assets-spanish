Parte 1: Evaluacion de Vulnerabilidad Personal

Responda cada pregunta honestamente para evaluar su vulnerabilidad a diferentes tipos de fraude.

1. Gestion de Contrasenas

Como gestiona sus contrasenas para diferentes cuentas?

2. Autenticacion de Dos Factores

Usa autenticacion de dos factores (2FA) para sus cuentas?

3. Habitos de Correo Electronico

Como maneja tipicamente los correos de remitentes desconocidos o con contenido sospechoso?

4. Privacidad en Redes Sociales

Como gestiona la configuracion de privacidad de sus redes sociales y el intercambio de informacion?

5. Monitoreo Financiero

Con que frecuencia revisa sus estados financieros e informes de credito?

6. Uso de WiFi Publico

Como usa las redes WiFi publicas?

7. Seguridad del Telefono

Como protege su telefono inteligente?

8. Respuesta a Contactos Inesperados

Como responde tipicamente a llamadas, textos o correos inesperados que dicen ser de su banco, una agencia gubernamental o soporte tecnico?

9. Actualizaciones de Software

Como maneja las actualizaciones de software en sus dispositivos?

10. Proteccion de Informacion Personal

Como maneja los documentos que contienen informacion personal sensible?

Parte 3: Recomendaciones de Proteccion Personalizadas

Recomendaciones de Seguridad de Cuentas

Para Alto Riesgo: Implemente un administrador de contrasenas para crear y almacenar contrasenas unicas y fuertes para todas las cuentas. Habilite la autenticacion de dos factores en todas las cuentas que lo ofrezcan, especialmente cuentas financieras y de correo electronico.

Para Riesgo Medio: Audite sus contrasenas existentes y actualice las que sean debiles o reutilizadas en multiples cuentas. Habilite la autenticacion de dos factores en sus cuentas mas sensibles.

Para Bajo Riesgo: Continue usando contrasenas unicas y 2FA. Considere usar llaves de seguridad de hardware para maxima proteccion en cuentas criticas.

Recomendaciones de Comportamiento en Linea

Para Alto Riesgo: Adopte una politica estricta de nunca hacer clic en enlaces de correos no solicitados. Evite realizar transacciones sensibles en WiFi publico. Siempre verifique independientemente la identidad de cualquiera que lo contacte sobre asuntos financieros.

Para Riesgo Medio: Pase el cursor sobre los enlaces para verificar destinos antes de hacer clic. Use una VPN cuando este en WiFi publico. Desarrolle un proceso de verificacion para comunicaciones inesperadas.

Para Bajo Riesgo: Mantenga su enfoque cauteloso. Mantengase informado sobre nuevas tacticas de phishing. Considere usar herramientas de filtrado de correo electronico para proteccion adicional.

Recomendaciones de Compartir Informacion

Para Alto Riesgo: Revise y restrinja la configuracion de privacidad de redes sociales. Limite la informacion personal compartida en linea. Comience a triturar documentos con informacion sensible. Sea selectivo con las solicitudes de amistad/conexion.

Para Riesgo Medio: Audite sus perfiles de redes sociales para eliminar detalles personales innecesarios. Establezca un sistema para almacenar o destruir de forma segura documentos sensibles.

Para Bajo Riesgo: Revise periodicamente su huella digital y continue minimizando el intercambio innecesario de informacion.

Recomendaciones de Monitoreo y Mantenimiento

Para Alto Riesgo: Configure revisiones semanales de cuentas financieras. Revise su informe de credito de las tres agencias. Habilite actualizaciones automaticas de software en todos los dispositivos.

Para Riesgo Medio: Cree un calendario para monitoreo financiero regular. Configure recordatorios para revisar su informe de credito tres veces al ano. Priorice las actualizaciones de software relacionadas con seguridad.

Para Bajo Riesgo: Considere usar servicios de monitoreo de credito para alertas en tiempo real. Mantenga su calendario de monitoreo regular.

Recomendaciones de Seguridad de Dispositivos

Para Alto Riesgo: Configure codigos de acceso fuertes y seguridad biometrica en todos los dispositivos. Habilite capacidades de borrado remoto. Instale y configure software de seguridad de buena reputacion.

Para Riesgo Medio: Fortalezca las medidas de seguridad existentes. Revise los permisos de aplicaciones en sus dispositivos. Habilite el bloqueo automatico de pantalla despues de cortos periodos de inactividad.

Para Bajo Riesgo: Mantenga las practicas de seguridad actuales. Considere proteccion avanzada como seguridad a nivel de aplicacion o encriptacion para datos sensibles.
